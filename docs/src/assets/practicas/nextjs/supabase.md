## Controlling Node version in a project

Here is an example of how to control the node version of your project with nvm and npm:

```bash
➜  supabase git:(master) cat .npmrc 
engine-strict=true%                                                                                  
➜  supabase git:(master) cat .nvmrc 
18
```
